## Title
Local File Inclusion (LFI) at SecureFile Viewer

## Vulnerability Type
Local File Inclusion (LFI)

## Summary  
The application at https://kzlabs.in/901.php is vulnerable to Local File Inclusion (LFI), allowing an attacker to manipulate file path parameters and access local files on the server. This may lead to the disclosure of sensitive information, such as configuration files, credentials, or application source code, and could facilitate further compromise of the system.

## Vulnerable Endpoint
https://kzlabs.in/091.php

## Steps to Reproduce  
1. Navigate to the following URL : `http://kzlabs.in/901.php`
2. Go to Enterprise Document Viewer Section.
3. In Input Box, if add payload `/etc/passwd` you will see sensitive data go leaked.


## Payload Used
`/etc/passwd`
