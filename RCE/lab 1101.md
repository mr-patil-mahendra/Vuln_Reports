## Title
Remote Code Execution (RCE) at NexusForge

## Vulnerability Type
Remote Code Execution (RCE)

## Summary  
The application at http://kzlabs.in/1101.php is vulnerable to Remote Code Execution (RCE), allowing an attacker to inject and execute arbitrary system commands on the underlying server. Successful exploitation could lead to unauthorized access, disclosure of sensitive information, modification of data, and complete compromise of the affected system.

## Vulnerable Endpoint
http://kzlabs.in/1101.php

## Steps to Reproduce  
1. Navigate to the following URL : `http://kzlabs.in/1101.php`
2. Go to command Section.
3. In `Command Configuration`, You will see the Input box for `command`
4. As i enter the   `ls` You see the server returns the Sensitive file’s contents.  


## Payload Used
`ls`
