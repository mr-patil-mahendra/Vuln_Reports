## Title
 XML External Entity (XXE) at SecureVault
 
## Vulnerability Type
XML External Entity (XXE)

## Summary  
The application in http://kzlabs.in/1406.php is vulnerable to XML External Entity (XXE) Injection, allowing an attacker to supply malicious XML input containing external entities. Successful exploitation may enable unauthorized access to sensitive files, disclosure of internal system information, server-side request forgery (SSRF), and potentially further compromise of the affected environment.
## Vulnerable Endpoint
http://kzlabs.in/1406.php

## Steps to Reproduce  
1. Navigate to the following URL : `http://kzlabs.in/1406.php`
2. Sign Up with New Credentail and Capture the request in Burpsuite or Caido.
3. take Request to repeater 
4. Add the payload at header of xml code  `<!DOCTYPE foo [ <!ENTITY xxe SYSTEM "file:///etc/passwd"> ]>` and this payload at any input value `&xxe;`
5. when You send the request you will see sensitive files in response.

## Payload
`<!DOCTYPE foo [ <!ENTITY xxe SYSTEM "file:///etc/passwd"> ]>`
`<text x="10" y="20">&xxe;</text>`
