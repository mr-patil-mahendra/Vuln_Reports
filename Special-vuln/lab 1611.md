## Title
Local File Inclusion (LFI) at U.S. Federal Document Portal

## Vulnerability Type
Local File Inclusion (LFI)

## Summary  
The application at https://kzlabs.in/1611.php is vulnerable to Local File Inclusion (LFI), allowing an attacker to manipulate file path parameters and access local files on the server. This may lead to the disclosure of sensitive information, such as configuration files, credentials, or application source code, and could facilitate further compromise of the system.

## Vulnerable Endpoint
https://kzlabs.in/1611.php

## Steps to Reproduce  
1. Navigate to the following URL : `http://kzlabs.in/1611.php`
2. You will See Document Retrival System Section.
3. In Input Box,  add payload `/etc/passwd` you will see sensitive data go leaked.


## Payload Used
`/etc/passwd`
