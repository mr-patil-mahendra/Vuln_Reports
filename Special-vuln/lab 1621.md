## Title
PII Disclosure via Publicly Accessible CDN Path

## Vulnerability Type
Information Disclosure / Sensitive Data Exposure

## Summary  
The application exposes sensitive PII through http://kzlabs.in/kzlabs/cdn/images/ahmetyuksek-winding-road-10313716_1920.jpg is a publicly accessible `/cdn/` path without proper access controls, allowing unauthorized users to access confidential information.

## Vulnerable Endpoint
http://kzlabs.in/kzlabs/cdn/images/ahmetyuksek-winding-road-10313716_1920.jpg

## Steps to Reproduce  
1. Navigate to the following URL : `http://kzlabs.in/kzlabs/cdn/images/ahmetyuksek-winding-road-10313716_1920.jpg`
2. Just go bach to directroy `http://kzlabs.in/kzlabs/cdn/`
3. You will see directory access. which can have senstive data.

## Impact
-  Unauthorized disclosure of sensitive personal information, leading to privacy violations and regulatory compliance risks.
-  Exposure of user data may facilitate identity theft, targeted phishing, or other malicious activities.

  
## Recommendations for fix

- Implement proper access controls and authentication to restrict access to sensitive files stored under the `/cdn/` path.
- Review and remove publicly accessible PII, and ensure sensitive files are stored in non-public locations with least-privilege permissions.
