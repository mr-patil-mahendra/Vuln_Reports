## Title
Local File Inclusion (LFI) WAF Bypass at AliyunCloud 

## Vulnerability Type
Local File Inclusion (LFI)

## Summary  
The application at https://kzlabs.in/1622.php is vulnerable to Local File Inclusion (LFI), allowing an attacker to manipulate file path parameters and access local files on the server. This may lead to the disclosure of sensitive information, such as configuration files, credentials, or application source code, and could facilitate further compromise of the system.

## Vulnerable Endpoint
https://kzlabs.in/1622.php

## Steps to Reproduce  
1. Navigate to the following URL : `http://kzlabs.in/1622.php`
2. Go to Cloud Shell Section
3. In Input Box, if add payload `tac /etc/passwd` you will see sensitive data go leaked.
4. Here `tac` bypass the WAF which was blocking tha `cat` command.


## Payload Used
`tac /etc/passwd`
