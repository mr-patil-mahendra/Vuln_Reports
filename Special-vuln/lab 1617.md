## Title
Default Credentials Authentication Bypass at cbse student databse portal

## Vulnerability Type
Default Credentials Authentication Bypass

## Summary  
The application at https://kzlabs.in/1617.php is vulnerable to Default Credentials Authentication Bypass, allowing an attacker to bypass the login page by try default credential

## Steps to Reproduce  
1. Navigate to the following URL : `http://kzlabs.in/1617.php`
2. In Email enter the admin mailid :  `admin@cbse.local` and in password `admin123`
3. As You click login , You have bypass and got access to database portal


## Password Used
`admin123`
